# Homebrew Tap: hazel

This is the Homebrew tap for Hazel.

## Install

```sh
brew tap flip-z/hazel
brew install hazel
```

## Maintenance

The formula in `Formula/hazel.rb` is intended to be updated automatically by GoReleaser from the main repo:

- `flip-z/hazel`
- `.goreleaser.yaml` `brews.repository` points here
